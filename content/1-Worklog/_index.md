This worklog documents the entire internship journey at **First Cloud Journey - AWS Study Group** over **12 weeks** (from 04/17/2026 to 07/10/2026).

The program is divided into 2 main phases: **8 weeks of learning and hands-on practice** across 21 AWS labs on cloudjourney.awsstudygroup.com, followed by **4 weeks of designing and deploying** the real-world project **ITCoach** — an AI-powered IT interview practice platform — on a Serverless AWS architecture. The website is live at https://itcoach24h.xyz.

**Week 1:** [Onboarding — Environment setup, tooling, Cloud Computing overview](1.1-week1/)

**Week 2:** [AWS Foundations — Account setup, IAM, VPC](1.2-week2/)

**Week 3:** [Compute — EC2, Auto Scaling, CloudWatch](1.3-week3/)

**Week 4:** [Storage & Database — S3, RDS, DynamoDB](1.4-week4/)

**Week 5:** [Networking & Security — Route 53, WAF, KMS](1.5-week5/)

**Week 6:** [DevOps — CloudFormation, CodePipeline, ECS](1.6-week6/)

**Week 7:** [Operations & Serverless — Lambda, Systems Manager, AWS Backup](1.7-week7/)

**Week 8:** [Advanced — EKS, Data Lake, SageMaker](1.8-week8/)

**Week 9:** [ITCoach Design — Serverless architecture, data model, deployment plan](1.9-week9/)

**Week 10:** [ITCoach Base Infrastructure — IAM, S3, DynamoDB, Cognito, SQS](1.10-week10/)

**Week 11:** [ITCoach Core Services — Lambda, API Gateway, CloudFront, WAF](1.11-week11/)

**Week 12:** [ITCoach Finalize — Route 53, ACM, Monitoring, Real code deployment](1.12-week12/)
